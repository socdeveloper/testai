# Образовательная платформа по Go, Redpanda, Reindexer, PostgreSQL и MongoDB

## Описание

Этот веб-сайт представляет собой образовательную платформу для изучения современного стека технологий:
- Go (Golang)
- Redpanda
- Reindexer
- PostgreSQL
- MongoDB

Платформа включает в себя:
- Инструкции по установке и настройке всех компонентов
- Уровни понимания для каждой технологии
- Практические задания с подсказками и решениями
- Дополнительные ресурсы для углубленного изучения

## Структура проекта

- `welcome.html` - Главная страница с приветствием и навигацией
- `index.html` - Страница с описанием технологий
- `requirements.html` - Страница с требованиями к кандидату
- `tasks.html` - Практические задания по каждой технологии
- `resources.html` - Дополнительные ресурсы и документация
- `styles.css` - Стили для веб-сайта
- `script.js` - JavaScript функциональность
- `README.md` - Данный файл с описанием проекта

## Особенности

- Адаптивный дизайн для различных устройств
- Интерактивные элементы (копирование команд, скрытие/показ информации)
- Плавная навигация между разделами
- Подробные инструкции по установке для каждой технологии
- Практические задания с разными уровнями сложности

## Требования к кандидату

Сайт также включает в себя подробное описание требований к кандидату на позицию разработчика с использованием этих технологий:
- Проактивный подход и высокая степень самостоятельности
- Системное и аналитическое мышление
- Уверенное владение Go и всеми перечисленными технологиями
- Опыт проектирования архитектуры и реализации сложных систем

## Запуск

Для запуска сайта локально достаточно открыть любой HTML-файл в браузере. Для размещения на веб-сервере поместите все файлы в корневую директорию сервера.
