Задания по Go

Задание 1: Создание простого веб-сервера

Создайте HTTP-сервер на Go, который отвечает "Hello, World!" на главной странице.

Подсказка

Используйте пакет net/http и функцию http.HandleFunc для обработки запросов.

Решение 
package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World!")
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}
